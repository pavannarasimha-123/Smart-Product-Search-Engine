# NLP-Based Product Search Ranking System Using TF-IDF and Machine Learning

## Overview

This project is an NLP-powered product search engine that ranks products based on user search queries using TF-IDF vectorization and Machine Learning techniques.

Users can enter a search query through a Flask web application and receive ranked product results.

---

## Features

* Product Search Engine
* TF-IDF Vectorization
* Cosine Similarity Ranking
* Random Forest Model
* XGBoost Benchmarking
* Flask Web Application
* Bootstrap User Interface

---

## Dataset

Home Depot Product Search Relevance Dataset

Files Used:

* train.csv
* product_descriptions.csv
